Alimentacion saludable para un peso saludable,

Es una pagina Web donde podras encontrar tip y recetas para poder entender y llevar una alimentacion saludable que puedas mantener y hacerla un estilo de vida.
Registrandote podras obtener un plan de alimentacion que puedes mantener a largo plazo.
Aprenderas a alimentarte y prepar tus comidas de forma facil, rapido, saludable, balanceado y sobreto delicioso.
Podras controlar las porciones de carbohidratos, proteinas, verduras y frutas para aprovechar al maximo sus nutrintes.

## FUNCIONALIDAD

Permite registrarte, deben crear un usuario, nombre, email y contraseña.
Permite iniciar sesion, deben ingresar usrio y contraseña.
Se guarda los usuarios en la base de dato mediante mongoose
